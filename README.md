# Vite React App

A React application built with Vite, TypeScript, Tailwind CSS, and shadcn/ui components.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router

## Deployment

This project is configured for deployment on Vercel. Simply connect your repository to Vercel and it will automatically detect the Vite configuration.
